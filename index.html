HTML is a coding language that stands for Hyper Text Markup Language and it describes the structure of a webpage using markup. CSS is a programming language that stands for Cascading Style Sheets, and it is a style sheet language used for describing the style of a webpage that has been written in a markup language such as HTML. Javascript is used for the reactivity of the webpage, such as the buttons and pop-ups. I have written the code on a program called Notepad ++.
